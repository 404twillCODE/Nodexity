## Contributing to Hexnode

Thanks for wanting to help.

### Before you start
- Check existing issues (or ask in Discord): `https://discord.gg/RVTAEbdDBJ`
- Keep PRs small and focused.

### Project structure
- `App/` — Electron + React desktop app (AGPL-3.0)
- `Website/` — Next.js marketing site (MIT)

### Development (App)
```bash
cd App
npm install
npm run dev
```

### Development (Website)
```bash
cd Website
npm install
npm run dev
```

### Reporting bugs
Open a GitHub issue with:
- What you expected vs what happened
- Steps to reproduce
- Screenshots/logs if applicable
- OS version and app version
